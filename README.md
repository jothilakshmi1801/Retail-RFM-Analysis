# Retail Customer Segmentation using RFM Analysis

##Project Overview

This project analyzes retail sales data to segment customers using RFM (Recency, Frequency, Monetary) analysis.

##Tools Used

* SQL (Oracle)
* Power BI
* Excel

##Project Steps

* Data cleaning (removed nulls and invalid values)
* Created TotalAmount column
* Calculated Recency, Frequency, Monetary metrics
* Assigned RFM scores using SQL
* Segmented customers into groups
* Built interactive Power BI dashboard

##Key Insights

* Best Customers contribute highest revenue
* Loyal Customers show consistent purchasing behavior
* At Risk Customers need re-engagement strategies

##Files

* rfm_final.csv
* RFM_Dashboard.pbix
* SQL Queries

##Outcome

This project helps businesses improve customer retention and maximize revenue through data-driven decisions.
